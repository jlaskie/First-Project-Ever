How to Use Fruit Tracker

Fruit Stored is the amount of fruit you have at home which has already been stored (leave at 0 if this is your first day picking fruit).


Fruit Picked is the amount of fruit you've picked today (leave at 0 if you haven't picked any fruit).


Fruit Eaten should be used if you ate any of the fruit you've picked or stored. This will be subtracted from your fruit total (leave at 0 if you didn't eat any fruit).
